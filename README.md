# Citation-Style-FPIK-UNDIP
Style Sitasi untuk Daftar Pustaka FPIK UNDIP untuk user MENDELEY
## 🛠️ Cara Install Custom Mendeley Style (Chicago Manual Modified)

Repositori ini berisi file `.csl` (Citation Style Language) kustom yang dimodifikasi sesuai dengan kebutuhan penulisan spesifik:
* **Format:** Chicago Manual of Style (Author-Date).
* **Tanpa Italics:** Judul buku dan jurnal tidak menggunakan huruf miring.
* **Italic *et al.*:** Penggunaan *et al.* dicetak miring dan tetap menggunakan bahasa Latin (bukan "dkk").
* **Pemisah:** Menggunakan titik sebagai pemisah utama antar elemen.

### 🚀 Cara Pemasangan di Mendeley Desktop / Reference Manager

Anda tidak perlu mengunduh file ini secara manual. Anda bisa langsung menautkan link **Raw** GitHub ke Mendeley agar selalu sinkron.

1.  **Dapatkan Link Raw:**
    * Klik file `Citation Style - FPIK UNDIP.csl` di repositori ini.
    * Klik tombol **"Raw"** di pojok kanan atas tampilan file.
    * Salin (Copy) URL yang muncul di browser (contoh: `https://raw.githubusercontent.com/username/repo/main/custom-chicago.csl`).

2.  **Buka Mendeley:**
    * Buka Mendeley Desktop atau Mendeley Reference Manager.
    * Pergi ke menu **View** > **Citation Style** > **More Styles...**

3.  **Install Style via URL:**
    * Pilih tab **"Get More Styles"**.
    * Tempelkan (Paste) URL Raw yang sudah disalin tadi ke kolom **"Download Style"**.
    * Klik **Download**.

4.  **Aktifkan di Microsoft Word:**
    * Buka Microsoft Word.
    * Masuk ke tab **Mendeley Cite** atau **References**.
    * Pilih **Citation Settings** > **Change Citation Style**.
    * Cari dan pilih `Custom Chicago Manual of Style (Italic et al - No dkk)`.
    * Klik **Update Citation Style**.

---

### 📝 Catatan Penting (Troubleshooting)

* **Masalah "dkk":** Jika kutipan masih memunculkan "dkk", pastikan pengaturan bahasa di Mendeley Word Plugin disetel ke **English (US)** atau **Default**. Style ini sudah dipaksa (*override*) menggunakan *et al.*, namun pengaturan dokumen Word terkadang menimpa aturan tersebut.
* **Update Style:** Jika file `.csl` di repositori ini diperbarui, Anda cukup mengulangi langkah "Download" di Mendeley untuk memperbarui format secara otomatis.
